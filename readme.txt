== Description ==



Plugin to create 50 state pages and one parent pages with links to the state pages organized in a table.




== Installation ==



1. Upload directory `ssm-state-builder` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Visit Plugins menu to add your post(s) or page(s).
